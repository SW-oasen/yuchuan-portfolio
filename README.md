## ğŸš€ Local Development

### Prerequisites
- Node.js 16+ installed
- Git installed

### Setup
```bash
# Clone the repository
git clone https://github.com/SW-oasen/yuchuan-portfolio.git
cd yuchuan-portfolio

# Install dependencies

# Start development server
```
